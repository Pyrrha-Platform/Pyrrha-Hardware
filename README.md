# Pyrrha hardware

This repository contains the [Pyrrha](https://github.com/Pyrrha-Platform/Pyrrha) solution hardware setup instructions which runs the [Pyrrha firmware](https://github.com/Pyrrha-Platform/Pyrrha-Firmware).
